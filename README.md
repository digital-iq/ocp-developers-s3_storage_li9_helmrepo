# Li9 S3 Storage Helm Repository

Public GitHub Pages Helm repository for Li9 S3 Storage charts.

Add the repository:

```bash
helm repo add li9-s3-storage https://digital-iq.github.io/ocp-developers-s3_storage_li9_helmrepo
helm repo update
```

Install the chart:

```bash
helm upgrade --install li9-s3 li9-s3-storage/li9-s3-storage \
  --namespace li9-s3 \
  --create-namespace \
  --version 1.1.0-alpha.15 \
  --set gateway.host=s3-gateway-li9-s3.apps.example.com
```

This repository is generated by the `ocp-developers-s3_storage_li9_helmchart`
pipeline. Do not edit chart packages or `index.yaml` by hand.

CI validates that `index.yaml` matches every packaged chart digest, that Helm
can consume the repository, and on `main` runs the published Pages chart through
the OpenShift install/AWS CLI/uninstall E2E suite from the Helm chart repo.
